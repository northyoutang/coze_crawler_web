## 📦 部署步骤

### 1. 安装依赖
```bash
cd coze_crawler_web
pip install -r requirements.txt
```

### 2. 安装浏览器（如未安装）
```bash
playwright install chromium
```

### 3. 清理旧数据库（可选但推荐）
```bash
rm -f data/*.db
```

### 4. 启动系统
```bash
python main.py
```

### 5. 访问 Web 界面
```
http://localhost:8000
```

## 🔧 系统架构

### 爬虫工作流程
```
1. 启动 Chromium 浏览器 (无头模式)
2. 访问 https://www.coze.cn/skills?tab=space
3. 查找搜索框并输入关键词
4. 提交搜索，监听 API 响应
5. 捕获 product/list 接口数据
6. 滚动页面触发分页加载
7. 持续捕获后续页面数据
8. 解析并保存为 Excel
9. 关闭浏览器
```

### API 路由（简化后）
- 主题词管理: GET/POST/PUT/DELETE `/api/keywords/*`
- 爬取控制: POST `/api/crawl/manual`, GET `/api/crawl/progress`, GET `/api/crawl/history`
- 定时设置: GET/POST `/api/scheduler/*`
- 文件管理: GET `/api/files/*`
- 认证: GET/POST `/api/auth/*`

## ⚠️ 注意事项

### 1. 浏览器资源
- 每个爬取任务启动一个浏览器实例
- 单个实例内存占用约 100-200MB
- 爬取完成后自动关闭释放资源

### 2. 爬取速度
- 浏览器页面加载需要时间
- 建议爬取间隔不小于 1 小时
- 可在设置中调整请求间隔

### 3. 网络要求
- 需要能够访问 `https://www.coze.cn`
- 稳定的网络连接

### 4. 数据库迁移
- 旧数据库包含 ms_token 字段不影响运行
- 但建议删除后让系统自动重建新表
